Lista de Definiciones

LISTAS
HTML ofrece varios mecanismos para elaborar listas de informacion, las cuales pueden contener elementos de los siguientes tipos: ordenados, no ordenados y definiciones.
LISTAS ORDENADAS
Las listas ordenadas muestran un conjunto de elementos o sucesos con un orden estricto. Se crean con los pares de etiquetas
LISTAS NO ORDENADAS
Las listas no ordenadas muestran un conjunto de elementos relacionados entre sí, pero sin ningún orden. Se crean con los pares de etiquetas
